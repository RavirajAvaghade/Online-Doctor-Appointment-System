 🏥 Hospital Management System

The **Hospital Management System** is a web-based application designed to manage hospital operations efficiently. It helps in handling patient records, doctor details, appointment scheduling, and administrative tasks in a streamlined way.

---

## 🚀 Features

* 🧑‍⚕️ Doctor Management (Add / View Doctors)
* 🧑 Patient Management (Registration & Records)
* 📅 Appointment Booking System
* 📖 Patient History Tracking
* 🔐 Secure Login & Authentication
* 🏥 Admin Dashboard
* 💻 Simple and User-Friendly Interface

---

## 🛠️ Technologies Used

* **Frontend:** HTML, CSS, Bootstrap
* **Backend:** Java (Servlets & JSP)
* **Database:** MySQL
* **Server:** Apache Tomcat

---

## 📂 Project Modules

* 👨‍⚕️ Doctor Module
* 🧑 Patient Module
* 🔐 Admin Module
* 📅 Appointment Management

---

## 🎯 Objective

The main objective of this project is to **digitize hospital operations**, reduce paperwork, and improve efficiency in managing patient and doctor information.

---

## ⚙️ Installation & Setup

1. Clone the repository

   ```bash
   git clone https://github.com/RavirajAvaghade/Online-Doctor-Appointment.git
   ```

2. Import project into **Eclipse / VS Code**

3. Configure **Apache Tomcat Server**

4. Setup MySQL Database

   * Create database
   * Import SQL file

5. Run the project on server
